 ## ▶ Como Executar Instruções básicas de como compilar/executar o código.

2. Acesse a pasta `src/html-css`.
3. Abra o arquivo `index.html`.
4. Execute no navegador utilizando a opção **Open with Live Server** ou clicando duas vezes no arquivo.
