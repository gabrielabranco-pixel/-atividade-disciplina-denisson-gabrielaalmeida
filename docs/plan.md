botão de download no canto direito

HERO SECTION 

A hero deve ter altura 90vh, divido em 3 áreas horizontais: 

Esquerda:
Apenas a headlin, sem ser grande demais, sem destaque ( o destaque é o print do app no meio) " O app perfeito para a hora do café"

Centro: 
Em destaque a imagem images/print_1.png que é um print do app

Direita: 
Uma lista de ponto-chave sobre o app:
- xícara de café 
- grãos de café 
- cafeteria 

## SEÇÂO DE FUNCIONALIDADE:

Vão ser usadas as imagens images/print_2.png, images/print_3.png e images/print_4.png que serão exibidas da seguinte forma:
- print / texto
- texto / print
- print / texto

Ou seja, irá alterar entre o print na esquerda e um texto explicativo na direita, vice-versa.

Descrição breve de cada print para você gerar um texto adequado:
- print_1.png = slogan café quente 
- print_2.png = cadastro de usuário
- print_3.png = página inicial ( opções de café)
